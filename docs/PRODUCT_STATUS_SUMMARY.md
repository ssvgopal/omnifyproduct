## ğŸ¯ Minimal Onboarding Steps

### **Step 1: User Registration** (2 minutes)
```bash
# 1. Visit application
curl -X POST http://localhost:8000/api/auth/register \
  -H "Content-Type: application/json" \
  -d '{"email": "client@example.com", "password": "secure123", "organization_name": "Client Corp"}'

# 2. Login and get token
curl -X POST http://localhost:8000/api/auth/login \
  -H "Content-Type: application/json" \
  -d '{"email": "client@example.com", "password": "secure123"}'
```

### **Step 2: Platform Integration** (5 minutes)
```bash
# 3. Connect GoHighLevel
curl -X POST http://localhost:8000/api/platforms/gohighlevel/connect \
  -H "Authorization: Bearer <token>" \
  -H "Content-Type: application/json" \
  -d '{"api_key": "ghl_api_key_here"}'

# 4. Connect LinkedIn Ads
curl -X POST http://localhost:8000/api/platforms/linkedin-ads/connect \
  -H "Authorization: Bearer <token>" \
  -H "Content-Type: application/json" \
  -d '{"client_id": "linkedin_client_id", "client_secret": "linkedin_secret"}'
```

### **Step 3: Feature Access** (1 minute)
```bash
# 5. Access EYES module for customer segmentation
curl -X GET http://localhost:8000/api/eyes/segments/30d \
  -H "Authorization: Bearer <token>"

# 6. Access analytics dashboard
curl -X GET http://localhost:8000/api/analytics/cross-platform \
  -H "Authorization: Bearer <token>"
```

### **Step 4: Automated Onboarding** (2 minutes)
```bash
# 7. Execute complete onboarding workflow
curl -X POST http://localhost:8000/revolutionary/client-onboarding \
  -H "Authorization: Bearer <token>" \
  -H "Content-Type: application/json" \
  -d '{
    "client_profile": {
      "name": "Client Corp",
      "industry": "ecommerce",
      "size": "medium"
    },
    "brand_assets": ["logo.png", "brand_guidelines.pdf"],
```
