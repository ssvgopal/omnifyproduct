## üéØ RECOMMENDED STACK BY USE CASE

### **DTC E-commerce Brands** ($5M-$150M revenue):

**Recommended Stack**:
1. **TripleWhale** (Primary) - Attribution & Analytics
2. **Klaviyo** (Secondary) - Lifecycle Marketing
3. **HubSpot** (Optional) - CRM if needed

**Why**:
- TripleWhale provides best-in-class attribution for Shopify brands
- Klaviyo is the DTC leader for lifecycle marketing
- HubSpot adds CRM if you need deal/pipeline management

---

### **Mid-Market B2B Companies** ($50M-$150M revenue):

**Recommended Stack**:
1. **HubSpot** (Primary) - CRM & Marketing Automation
2. **TripleWhale** (Optional) - If doing paid advertising
3. **Klaviyo** (Optional) - If doing email marketing

**Why**:
- HubSpot is the market leader for mid-market CRM
- TripleWhale adds attribution if running paid ads
- Klaviyo adds lifecycle marketing if needed

---

### **Small Businesses / Agencies**:

**Recommended Stack**:
1. **GoHighLevel** (Primary) - All-in-one solution
2. **HubSpot** (Optional) - If outgrowing GoHighLevel

**Why**:
- GoHighLevel is designed for SMB/agency use cases
- All-in-one solution reduces complexity
- Lower cost than separate platforms

## üöÄ QUICK START GUIDE

### **For New DTC Brands**:

1. **Start with TripleWhale**:
   - Connect Shopify store
   - Connect paid social accounts (Meta, Google, TikTok)
   - Get attribution data flowing
   - Enable Omnify MEMORY module for ROI insights

2. **Add Klaviyo** (if doing email/SMS):
   - Connect Klaviyo account
   - Sync customer lists
   - Set up lifecycle flows
   - Enable Omnify CURIOSITY module for recommendations

3. **Add HubSpot** (if needed for CRM):
   - Connect HubSpot account
   - Import contacts
   - Set up sales pipeline
   - Enable Omnify CURIOSITY module for campaign recommendations

---

### **For Existing GoHighLevel Users**:

**Option 1: Keep GoHighLevel** (if it works for you):
- Continue using GoHighLevel
- No migration needed
- GoHighLevel remains fully supported

**Option 2: Migrate to New Stack** (recommended for DTC brands):
- See `/docs/GOHIGHLEVEL_MIGRATION_GUIDE.md`
- Migrate to TripleWhale/HubSpot/Klaviyo
- Get better attribution, analytics, and predictive intelligence
